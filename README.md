# ExtractPDF4J Site (GitHub Pages ready)

- Vite `base: './'`
- Outputs to `/docs`
- Includes `.nojekyll` in build (copied from `public/`)

## Run
npm install
npm run dev

## Build
npm run build   # -> ./docs

## Publish on GitHub Pages
- Push to repo
- Settings → Pages → Source: **main** / **/docs**
- Done
